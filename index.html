HealthAssist

AI-powered healthcare platform built from scratch and shipped to production as a solo project.

Next.js · Node.js · Express · Firebase · Prisma · Three.js · Python

Designed the full stack end-to-end: polished web frontend, REST API, realtime data layer, and deployment flow.

Built a custom interactive 3D body map where patients tap body regions to start symptom triage, without relying on an off-the-shelf widget.

Integrated a Telegram bot as an alternative client on the same backend with zero code duplication.

Implemented stateless QR-based sessions with automatic deletion on session end for safer patient flow handling.

Validated the product with 70 healthcare stakeholders and used reliability and privacy concerns to shape the architecture and UX.
